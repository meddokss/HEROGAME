# Games Hero
Создание героя.
Возможность одевать вещи из рюкзака / снимать вещи в рюкзак.
Одетые вещи дают новые статы герою - увеличивая или снижая их.
Статы отражены в отдельном окне., в зависимости от типа оружия, вещей - изменяются.
